# Zeta Android App 🚀

Aplikasi Android yang dibuat dengan **Python + BeeWare**

## Fitur
- UI Modern dengan BeeWare/Toga
- Support Android (APK)
- Build otomatis via GitHub Actions

## Download APK
Ambil APK terbaru di [Releases](https://github.com/${{ github.repository }}/releases)

## Development

### Install Dependencies
```bash
pip install briefcase toga
```

### Local Build (Linux/Windows with Android SDK)
```bash
briefcase create android
briefcase build android
```

### Run in Dev Mode
```bash
briefcase dev android
```

## Tech Stack
- Python 3.10+
- BeeWare Toga
- Briefcase
